# Decision Types

## Type 1: One-Way Door (Hard to Reverse)

These decisions are difficult or costly to undo once made.

**Examples:**
- Major process changes
- Technology platform selection
- Org structure changes
- Long-term vendor commitments
- Policy changes with broad impact

**Approach:**
- Thorough analysis before deciding
- Multiple stakeholder input
- Executive approval required
- Full 6-step procedure (see procedure.md)
- Document rationale carefully — this is a decision record worth keeping

---

## Type 2: Two-Way Door (Reversible)

These decisions can be undone or adjusted based on what you learn.

**Examples:**
- Process experiments
- Pilot programs
- Tool trials
- Incremental improvements
- Workflow tweaks

**Approach:**
- Decide fast, learn fast, iterate
- Don't over-analyze
- Set a clear success signal and review date
- Lightweight documentation — capture enough to learn from it

---

## How to Determine the Type

Ask: "If this turns out to be wrong, how hard is it to go back?"

- Hard/expensive/relationship-damaging to reverse = Type 1
- Easy/cheap/low-stakes to reverse = Type 2

Common mistake: treating Type 2 decisions like Type 1. Analysis paralysis on reversible decisions wastes time and signals poor judgment. When in doubt, ask whether you can course-correct in 30-60 days without major cost.
